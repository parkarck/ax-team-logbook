01 · Agent

일반 AI vs AI 에이전트

일반 AI 챗봇

💬 질문에만 답함
📖 정보 제공
🔄 수동적 대화
❓ "어떻게 해요?"

→
AI 에이전트

🛠️ 직접 작업 수행
🎯 목표 달성
🚀 능동적 행동
✅ "이렇게 해드렸어요"

실제 예시
"웹사이트 성능 최적화해주세요"
일반 AI → 방법만 설명  |  AI 에이전트 → 직접 분석·코드 수정·테스트·보고서 작성
🧠 언어 모델
  • 추론 및 계획
  • 상황 이해
  • 의사결정
🛠️ 도구
  • 파일 시스템
  • 웹 검색
  • API 호출
  • 코드 실행
🎯 자율성
  • 자율적 실행
  • 피드백 루프
  • 결과 도달
01 · Agent

에이전트 아키텍처 유형 (2025)

1단계 · 단일 에이전트
  • 간단한 작업, 프로토타입에 적합
  • 빠른 설정, 낮은 비용
  • 복잡한 작업에는 한계
2단계 · 계층형
  • 관리자 + 전문가 팀 구조
  • 전문성 분리, 확장성 확보
  • 플래너 → 리서처 → 라이터 → QA
3단계 · 연합 네트워크
  • 독립적 에이전트들이 협력
  • 최대 확장성, 병렬 처리
  • 2025-2026년 주류 패턴
핵심 공식   Agent = LLM + Tools + Autonomy
에이전트는 단순히 답을 생성하는 것이 아니라, 목표를 향해 자율적으로 행동합니다.

02 · MCP

MCP란 무엇인가?

MCP 이전

AI ──→ GitHub 전용 연결
AI ──→ Slack 전용 연결
AI ──→ Database 전용 연결
AI ──→ File System 전용 연결
= 각각 따로 개발 필요 ❌

→
MCP 이후

AI ──→ MCP ──→ GitHub
            ├────→ Slack
            ├────→ Database
            └────→ File System
= 하나의 표준으로 모두 연결 ✅

표준화
  • 단일 표준 프로토콜
  • 재사용 가능한 커넥터
  • 확장 용이
상황 공유
  • 데이터 + 의미(Context) 전송
  • 전년 대비 증감, 원인까지 전달
  • 깊이 있는 AI 판단 가능
양방향 통신
  • 읽기: AI가 데이터 수집
  • 쓰기: AI가 직접 작업 수행
  • 실시간 상호작용
02 · MCP

MCP 도입 현황 (2025)

회사 도입 현황 활용 사례
Anthropic창시자Claude 전면 탑재 (2024.11 오픈소스 공개)
OpenAI채택Agents SDK에 통합 (2025.3)
Google전면 지원Gemini 탑재, Maps·Cloud DB 공개
Microsoft도입 중Agentic AI Foundation 공동 설립, Azure 통합 예정
Hugging Face통합 중프레임워크 통합, 개발자 도구 강화
결론: MCP는 2025년 현재 사실상 업계 표준(de facto standard)이 되었습니다. AI와 외부 시스템을 연결할 때 MCP를 쓰면 한 번 구현으로 모든 AI에 연결 가능합니다.

03 · Skills

스킬 구조와 3단계 로딩

📁 스킬 폴더 구성
  • SKILL.md (필수) — 메타데이터 + 지시사항
  • scripts/ (선택) — Python, Bash 스크립트
  • references/ (선택) — 참조 문서
  • assets/ (선택) — 템플릿, 폰트, 아이콘
1단계 — 항상 로드
  • YAML 프론트매터
  • 스킬 이름 · 설명
  • "언제 사용할지" 판단 정보
2단계 — 관련 시 로드
  • SKILL.md 본문 전체
  • 전체 지시사항 · 상세 가이드
3단계 — 필요시 로드
  • 참조 문서 · 스크립트 · 추가 자산
03 · Skills

프롬프트 vs 스킬 vs 서브 에이전트

특징 📝 프롬프트 📦 스킬 🤖 서브 에이전트
구조텍스트 지시사항파일 시스템 폴더독립적 AI 인스턴스
맥락매번 반복 입력자동 로드독립적 컨텍스트
전문성일반적도메인별 패키지깊이 있는 특화
용도빠른 수정재사용 가능한 표준복잡한 작업 위임
예시"React로 작성"React 스킬 폴더Frontend 서브 에이전트
팀 컨벤션도 스킬로!
API 설계 표준, 코드 리뷰 체크리스트, 테스트 전략 등 팀의 노하우를 스킬로 패키징하면 모든 에이전트가 동일한 기준으로 작업합니다.

05 · Scenarios — 시나리오 1

🛒 이커머스 플랫폼 개발 ShopAI

상황: 스타트업 ShopAI, 3개월 내 MVP 출시 목표 · 프론트엔드 2명 · 백엔드 2명 · 2025년 Q1
서브 에이전트 팀
  • Frontend — React/Next.js 스킬, UI/UX
  • Backend — Node.js/Express, RESTful API
  • Database — PostgreSQL 스키마, ORM
  • Security — OWASP Top 10, 인증/인가
  • QA — E2E 테스트, 성능 테스트
MCP 연동
  • GitHub MCP — 코드 저장소 · PR 검토
  • Slack MCP — 팀 알림 자동화
  • Database MCP — 제품 데이터 · 재고
  • Payment MCP — 결제 · 주문 처리
결과
  • 개발 기간 12주 → 8주 (33% 단축)
  • 버그 밀도 40% 감소, 커버리지 85%
  • 초기부터 보안 고려 → 취약점 0건
05 · Scenarios — 시나리오 2 · 3

🏦 금융 리스크 FinBank & 🏥 헬스케어 MedAI

대형 은행, AI 기반 실시간 리스크 분석 · 규제 준수
서브 에이전트
  • Data Analyst — 거래 패턴 · 이상 탐지
  • Compliance — Basel III · 보고서
  • Security Auditor — 감사 로그 분석
  • Model Validator — ML 모델 검증
결과
  • 오탐율 60%, 미탐율 45% 감소
  • 규제 보고 시간 80% 단축
  • 운영 비용 40% 절감
대학병원, AI 기반 진단 보조 · HIPAA 준수 · 의료기기 인증
서브 에이전트
  • Medical Imaging — X-ray/CT/MRI
  • Lab Results — 혈액 검사 판독
  • Medical Records — EHR 분석
  • Clinical Decision — 치료 계획 제안
  • Privacy Officer — HIPAA 준수
결과
  • 초기 진단 정확도 92%
  • 문서 작성 70%, 영상 판독 50% 단축
  • 약물 상호작용 경고 100% 감지

🚀 2025년 트렌드 & 실습 과제
